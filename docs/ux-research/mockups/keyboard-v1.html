Jot iOS — Keyboard Mockup

V1 · Pure Mic Minimum-viable

Single, very-large mic button centered on an otherwise empty keyboard plane. Nothing else competes for attention. The keyboard is one job: tap to dictate. The 3 preset prompts (Articulate · Shorter · Professional) only appear when the host actually has a selection — surfaced as a thin segmented row above the mic. The point: prove the absolute-minimum keyboard works before adding chrome. Borrows mic placement from Wispr Flow; borrows monochrome restraint and the warm-resume ring treatment from Superwhisper.

V1 trade-offs:

  • Pro: impossible to confuse — the affordance IS the keyboard. Smallest test matrix.
  • Pro: recording state lives entirely in mic color/glyph swap; no separate status zone to maintain.
  • Con: no in-keyboard feedback during recording (no waveform, no elapsed time) — user has to trust the system.
  • Con: the "tap mic to dictate" mental model is undocumented inside the surface itself; relies on first-run onboarding card carrying the load.
