Design notes

  • Three labelled buttons in one row: Magic (✦, left) → Speak (🎙, center, primary) → Actions (⋯, right). Speak stays the visual anchor (blue, fills the middle).
  • Tap = default action (Magic → "Rewrite" prompt; Actions → Copy if selection, else Paste).
  • Long-press opens an iOS-native context menu (.contextMenu) — system-styled, blurred, animated.
  • Magic stays disabled until there's a recent transcript in the .followUp window after dictation; then it lights blue.
  • Subtle horizontal dash beneath Magic / Actions = long-press affordance ("there's more here").
  • Disabled rows in menus stay visible with a hint ("· no selection") instead of disappearing — preserves spatial memory.
  • The default row in each menu is marked with a left blue rail + bold weight, so users learn what the tap shortcut does.
